# 修改管理员账号和密码指南

## 方法一：通过管理后台修改密码（推荐）

1. 访问管理后台登录页面：`http://你的网址/admin/login`
2. 使用当前管理员账号登录（默认：admin / admin123）
3. 登录后自动跳转到管理后台首页
4. 点击页面上方的「**修改密码**」按钮
5. 输入当前密码、新密码、确认新密码
6. 点击「确认修改」

## 方法三：修改默认管理员账号（代码层面）

如果你想要修改默认的管理员用户名和密码，可以编辑以下文件：

**文件位置：** `app/__init__.py`

**找到这段代码（约第 112-116 行）：**
```python
# 创建默认管理员账号
if Admin.query.count() == 0:
    admin = Admin(username='admin')
    admin.set_password('admin123')
    db.session.add(admin)
```

**修改为：**
```python
# 创建默认管理员账号
if Admin.query.count() == 0:
    admin = Admin(username='你的用户名')  # 修改用户名
    admin.set_password('你的密码')        # 修改密码
    db.session.add(admin)
```

**注意：** 这个修改只会在数据库中没有管理员账号时生效（首次部署）。如果已经部署过，请使用方法一或方法二。

## 方法四：通过 CloudBase 环境变量（推荐生产环境）

可以在 CloudBase 控制台配置环境变量来设置默认管理员密码：

1. 登录 [腾讯云 CloudBase 控制台](https://console.cloud.tencent.com/tcb)
2. 选择你的云托管服务
3. 进入"环境变量"配置页面
4. 添加环境变量：
   - `DEFAULT_ADMIN_USERNAME`：管理员用户名
   - `DEFAULT_ADMIN_PASSWORD`：管理员密码
5. 重新部署应用

**注意：** 需要在 `app/__init__.py` 中修改代码来支持从环境变量读取默认管理员凭据。

## 安全建议

1. **不要使用默认密码**：默认密码 `admin123` 非常不安全，请立即修改
2. **使用强密码**：至少 8 位，包含大小写字母、数字和特殊字符
3. **定期更换密码**：建议每 3-6 个月更换一次管理员密码
4. **不要泄露密码**：管理员账号可以删除所有数据，请妥善保管

## 常见问题

### Q1：忘记管理员密码怎么办？
**A：** 使用方法二（直接修改数据库）重置密码。

### Q2：修改密码后无法登录？
**A：** 确认新密码至少 6 位，并且两次输入的确认密码一致。

### Q3：可以创建多个管理员账号吗？
**A：** 当前系统只支持一个管理员账号。如果需要多管理员，需要修改代码。

### Q4：如何删除管理员账号？
**A：** 当前系统不支持删除管理员账号，至少保留一个。
